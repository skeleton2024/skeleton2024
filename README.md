## Selected Work

### Multimodal Physiological Stress Recognition

Multimodal physiological stress recognition on WESAD with ECG / EDA / RESP / TEMP signals.

- Built a 60s-window / 30s-step feature pipeline with 37 physiological features.
- Used Leave-One-Subject-Out validation to reduce subject leakage.
- Compared LR / RF / AdaBoost / XGBoost.
- Achieved XGBoost Accuracy 0.930 / Macro F1 0.921.
- Integrated a deployed Random Forest model with LabVIEW Python Node.

Repo: [multimodal-physio-stress-recognition](https://github.com/skeleton2024/multimodal-physio-stress-recognition)

### TeamMind / team-brain

Local-first AI context agent for turning sources into signals, memories, actions, briefs, and result feedback loops.

- Built context intake, signal extraction, memory extraction, and action suggestion flows.
- Designed a source -> signal -> memory / action -> brief / result -> update loop.
- Added demo data, localStorage hardening, smoke tests, and iterative PRs.

Repo: [team-brain](https://github.com/skeleton2024/team-brain)
